Version 4.0 - One data input added to all journeys

Top level ideas that were discussed during the initial review of the live version

  • Updating the homepage to reflect the Guided Assistance UR - e.g. moving the start now button underneath the "Before you start" information
  • Adding the ability to create a basic account (Username/Password) to enable Users to save their progress - as applying is complicated and there could be many delays/hurdles for the User to overcome
  • Helping to set expectations - this would require relocating the summary page of what's been completed, to closer in the User's journey
  • Grouping form fields where possible - this would enable the User to take fewer steps during signup
  • Improving the UX by linking out to all forms/documents that are mentioned
